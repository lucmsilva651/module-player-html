# Chiptune WebPlayer (module-player-js)
This is a Web player for chiptunes, based on the [chiptune3](https://github.com/drsnuggles/chiptune) library.

## Dependencies
To install dependencies:

```bash
bun install
```

To build:

```bash
bun run build
```

To build and watch for changes:
```bash
bun run dev
```
